# Dota 2 Boot Breaker Auto-Player (Dark Carnival Mini-Game)

Автоматический помощник для прохождения мини-игры **Boot Breaker** (Арканоид) в событии **Dark Carnival** (Тёмный карнавал) в Dota 2. 

Скрипт использует компьютерное зрение (OpenCV) для отслеживания движения сапога и платформы в реальном времени, рассчитывает траекторию отскока от стен и автоматически направляет каретку для поимки. Также поверх игры выводится удобный информационный оверлей.

---

## ⚠️ Важные требования перед запуском

Чтобы скрипт работал корректно, необходимо соблюдать три условия:
1. **Разрешение экрана в Dota 2**: строго **1920x1080**. Режим отображения — *«В окне без рамки»* (Borderless Windowed).
2. **Запуск от имени Администратора**: операционная система Windows блокирует отправку нажатий клавиш в Dota 2, если скрипт запущен в обычном режиме.
3. **Операционная система**: Windows (скрипт использует Win32 API для симуляции ввода клавиатуры).

## 🚀 Инструкция по установке и запуску (для новичков)

### Шаг 1. Установка Python
Если у вас еще не установлен Python:
1. Перейдите на официальный сайт [python.org](https://www.python.org/downloads/) и скачайте последнюю версию для Windows.
2. При установке **обязательно** поставьте галочку напротив пункта **"Add Python to PATH"** (Добавить Python в PATH) в нижней части установщика. Это критически важно.
3. Завершите установку.

### Шаг 2. Скачивание проекта
1. Нажмите зеленую кнопку **Code** вверху этой страницы GitHub и выберите **Download ZIP**.
2. Распакуйте архив в любую удобную папку (например, на Рабочий стол).

### Шаг 3. Установка библиотек
1. Откройте командную строку Windows. Для этого нажмите комбинацию клавиш `Win + R`, введите `cmd` и нажмите Enter.
2. Перейдите в папку с распакованным скриптом. Например, если папка находится на рабочем столе:
   ```bash
   cd Desktop/dota2-boot-breaker-main
   ```
3. Установите необходимые библиотеки одной командой:
   ```bash
   pip install -r requirements.txt
   ```

### Шаг 4. Запуск скрипта
1. Закройте текущую командную строку.
2. Откройте командную строку **от имени Администратора**: нажмите кнопку «Пуск», введите в поиске `cmd`, нажмите правой кнопкой мыши на «Командная строка» и выберите **«Запуск от имени администратора»**.
3. Снова перейдите в папку со скриптом (укажите ваш полный путь, если папка на рабочем столе):
   ```bash
   cd C:\Users\ИМЯ_ПОЛЬЗОВАТЕЛЯ\Desktop\dota2-boot-breaker-main
   ```
4. Запустите скрипт командой:
   ```bash
   python main.py
   ```

## 🎮 Как пользоваться в игре

1. Запустите Dota 2, зайдите в событие и откройте мини-игру **Boot Breaker**.
2. Запустите скрипт (если еще не запустили). Поверх игры появится полупрозрачный интерфейс (оверлей) с рамкой и HUD-панелью.
3. Скрипт начнет автоматически двигать платформу влево-вправо. При появлении стартового экрана он сам нажмет пробел для запуска раунда.
4. **Горячие клавиши:**
   * **Удержание `SHIFT`**: ставит скрипт на паузу и возвращает вам ручной контроль над платформой. Как только вы отпустите `SHIFT`, авто-помощник продолжит работу.
   * **Клавиша `ESC`**: нажмите её, предварительно кликнув по оверлею, чтобы полностью закрыть программу.

## 🛠️ Возможные проблемы и их решение

* **Каретка (платформа) не двигается, хотя оверлей видит сапог**
  * *Решение*: Вы забыли запустить командную строку от имени Администратора. Закройте консоль и запустите `cmd` заново с правами администратора.

* **Оверлей или рамка смещены/отображаются не на мини-игре**
  * *Решение*: Проверьте разрешение экрана в настройках Dota 2. Оно должно быть ровно 1920x1080.

* **Игра сильно тормозит или падает FPS**
  * *Решение*: В скрипте установлено ограничение в 120 FPS для снижения нагрузки на процессор. Если ваш компьютер перегружен, вы можете снизить частоту кадров в игре (например, в консоли Dota 2 прописать `fps_max_ui 30`, что также замедлит полет сапога и упростит его поимку).
